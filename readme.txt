# Express Exercise

This repository contains a express exercise aimed at learning and practicing basic express features. 

Based on the provided instructions for your web application project, here's a README file draft that outlines the project's goals, setup instructions, and usage:

Overview
This project is a web application built using Express that contains three pages: Home, Our Services, and Contact Us. The application is designed to be accessible only during working hours (Monday to Friday, 9 AM to 5 PM). It includes basic styling with CSS and optionally utilizes a template engine for rendering views.

Features
Three Pages:

Home Page: The landing page of the application.
Our Services: Information about the services offered.
Contact Us: A form or details for contacting the company.
Navigation:

Each page includes a navigation bar with links to the Home, Our Services, and Contact Us pages.
Working Hours Restriction:

The web application is accessible only during working hours: Monday to Friday, 9 AM to 5 PM.
Custom Middleware:

A custom middleware is implemented to check the time of the request and restrict access outside of working hours.
Styling:

The pages are styled with CSS.
Optional Template Engine:

Use of a template engine (e.g., EJS, Pug, Handlebars) is optional for rendering views.
